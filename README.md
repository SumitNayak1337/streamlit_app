## ✨ Key Features

- 🧠 **Custom Knowledge Base**: NEXUS is trained on Sumit's proprietary cybersecurity checklists and portfolio.
- 🖼️ **Screenshot Intelligence**: Upload images of findings, recon data, or vulnerability scans—NEXUS will parse and analyze them with high precision.
- 📄 **Auto-Generated DOCX Reports**: Select from Offensive (Pentest) or Defensive (SOC) templates to generate high-quality compliance-ready reports.
- 🔍 **Real-time Tactical Guidance**: Get exact commands, payloads, and remediation steps mapped to Sumit’s expert checklists.
- ⚡ **Lightning Fast Analysis**: Powered by the latest **Gemini Pro Vision** (genai) SDK for state-of-the-art results.

## 🔑 How to Get Your Free API Key

NEXUS uses Google's Gemini models for its intelligence. You can get your own API key for free in less than a minute!

1.  Visit **[Google AI Studio](https://aistudio.google.com/app/apikey)**.
2.  Sign in with your Google account.
3.  Click on the **"Create API Key"** button.
4.  Copy the generated key and paste it into the NEXUS sidebar.

## 🛠️ Installation & Setup

To run NEXUS locally, follow these simple steps:

### 1. Clone the repository
```bash
git clone https://github.com/SumitNayak1337/streamlit_app.git
cd streamlit_app
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Launch the App
```bash
streamlit run app.py
```
